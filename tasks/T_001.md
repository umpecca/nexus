# T_001-Application Shell

**Task Name:** Application Shell
**Task ID:** T_001
**Task Description:** Create the initial NW.js, Vite, React, and TypeScript application shell for Nexus.
**Task Status:** CLOSED
**Task Status Details:** Added package metadata, Vite configuration, TypeScript configuration, HTML entry point, React bootstrap, and global styling.

## Problem

The repository describes a desktop Markdown editor but does not yet contain a runnable application.

## Solution

1. Add package scripts and dependencies for React, Vite, TypeScript, MDXEditor, and NW.js.
2. Add the Vite HTML entry point.
3. Add the React bootstrap file.
4. Add a focused application shell with toolbar, workspace, and side panel.

## Verfication

Run `npm install`, then `npm run dev` to start the Vite development server. Run `npm run build` to verify the TypeScript and production bundle.
