# T_082: Add profile-scoped paragraph spacing settings

## Status

Done

## Summary

Add a paragraph spacing preference that lets users tune the vertical gap after paragraphs in the editor and rendered exports.

## Implementation Notes

- Extended profile-scoped settings with `paragraphSpacingPixels`, defaulting to 16px.
- Added a unit-labeled Paragraph spacing field to the settings dialog.
- Applied paragraph spacing to rich-text paragraph rendering through an editor CSS variable.
- Passed paragraph spacing through HTML export, PDF export, and print preview rendering.
- Added unit tests for default, valid saved, invalid saved, and persisted paragraph spacing settings.
