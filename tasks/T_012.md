# T_012-Electron File Menu Exit Action

**Task ID:** T_012

**Task Description:** Add a File/Exit item to the Electron app menu and implement application quit behavior.

**Status:** CLOSED

## Acceptance Criteria

1. The Electron File menu includes an Exit item.
2. Selecting File/Exit quits the Electron application.
3. The menu item has a standard keyboard accelerator.

## Implementation Notes

- Added a separator and Exit item to the Electron File menu template.
- Wired Exit directly to `app.quit()` in the Electron main process.

## Verification

Run `npm run build`.
