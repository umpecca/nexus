# T_002-Markdown Editing Workflow

**Task Name:** Markdown Editing Workflow
**Task ID:** T_002
**Task Description:** Add visual and source Markdown editing modes with local draft persistence.
**Task Status:** CLOSED
**Task Status Details:** Added MDXEditor-based visual editing, textarea source editing, local storage helpers, and default draft content.

## Problem

Users need a simple way to start editing Markdown immediately and recover local draft work between sessions.

## Solution

1. Store the current Markdown document in React state.
2. Render visual editing through MDXEditor.
3. Render source editing through a plain textarea fallback.
4. Persist draft state to local storage.
5. Restore draft state on launch.

## Verfication

Run `npm run dev`, edit the document, refresh the app, and confirm the draft remains available.
