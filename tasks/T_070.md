# T_070: Show units for numeric settings values

**Status:** Done

**Task Description:** Make the Preferences dialog show the units behind numeric editor settings so font size, paper dimensions, and margins are not ambiguous.

## Acceptance Criteria

1. Base font size shows a `px` unit next to its numeric value.
2. Paper size options show their dimensions in inches.
3. Margin fields show an `in` unit next to each numeric value.
4. PDF export keeps using the same inch-based margin values shown in Preferences.

## Implementation Notes

- Added unit suffixes to the settings dialog numeric controls.
- Included paper dimensions in the paper-size select option labels.
- Removed PDF margin conversion so Preferences and export use the same inch-based values.
