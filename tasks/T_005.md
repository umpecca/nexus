# T_005-Remove Deferred Panels

**Task Name:** Remove Deferred Panels
**Task ID:** T_005
**Task Description:** Remove the visible changed-lines and inline AI sections from the current app shell.
**Task Status:** CLOSED
**Task Status Details:** Removed the changes mode, side panel, baseline action, inline AI command form, diff utility, and related styles. Updated the product and architecture documents to defer these capabilities.

## Problem

The initial scaffold exposed changed-lines review and inline AI controls before the core editor workflow was ready for those concepts.

## Solution

1. Remove the Changes tab and changed-lines panel from the editor UI.
2. Remove the Inline AI section and local AI note behavior.
3. Remove baseline-specific UI and local storage state.
4. Update documentation so changed-lines review and inline AI are future work.
5. Keep visual editing, source editing, import, export, and local draft persistence intact.

## Verfication

Run `npm run build` and `npm run test`. Open the app and confirm only the Visual and Source editor modes are visible, with no changed-lines or inline AI sections.
