# T_076: Add landscape paper orientation

## Status

Done

## Summary

Add a profile-scoped paper orientation preference so the rich-text paper preview and PDF export can use portrait or landscape layout.

## Implementation Notes

- Added a `pageOrientation` setting with Portrait and Landscape options.
- Swapped editor paper width and height when Landscape is selected.
- Passed the selected orientation through PDF export and Electron's print flow.
- Preserved portrait as the default for existing and invalid saved settings.

## Verification

- Unit tests cover default, valid, invalid, and saved orientation settings.
- Run `npm run test`.
- Run `npm run build`.
