# T_067: Add a gray bottom border to the rich-text toolbar

**Status:** Done

**Task Description:** Add a subtle bottom border to the toolbar using the same gray line color as the toolbar button groups.

## Acceptance Criteria

1. Rich-text toolbar shows a bottom border.
2. The toolbar bottom border uses `var(--line)`, matching the toolbar group borders.
3. Source and diff floating view controls do not show an extra toolbar bar border.

## Implementation Notes

- Added `border-bottom: 1px solid var(--line)` to the MDXEditor toolbar shell.
- Preserved `border-bottom: 0` when the toolbar is in floating source/diff mode.
