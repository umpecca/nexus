# T_008-Enable Broad MDXEditor Toolbar

**Task Name:** Enable Broad MDXEditor Toolbar
**Task ID:** T_008
**Task Description:** Add every practical built-in MDXEditor toolbar item by using MDXEditor's Kitchen Sink toolbar and enabling the supporting plugins.
**Task Status:** CLOSED
**Task Status Details:** Replaced the minimal toolbar with `KitchenSinkToolbar` and enabled link, image, table, thematic break, code block, CodeMirror, admonition directive, frontmatter, Sandpack, Markdown shortcut, and diff/source plugins.

## Problem

The editor toolbar only exposed undo/redo, bold, italic, underline, and source mode, leaving many MDXEditor capabilities unavailable.

## Solution

1. Use MDXEditor's built-in `KitchenSinkToolbar` to expose the broadest toolbar set available from the package.
2. Enable the feature plugins required by the toolbar buttons.
3. Add a small local Sandpack preset so the Sandpack insert control has a valid configuration.
4. Keep the toolbar inside the MDXEditor surface and let it wrap when many controls are visible.
5. Update product and architecture docs to describe the broad built-in toolbar.

## Verfication

Run `npm run build` and `npm run test`. Open the app and confirm the MDXEditor toolbar shows the expanded set of formatting, block, insert, and source controls.
