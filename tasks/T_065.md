# T_065: Float the view controls over source and diff modes

**Status:** Done

**Task Description:** Make the right-side editor view controls overlap the Markdown source and diff editors at the top-right instead of reserving a full toolbar row in those modes.

## Acceptance Criteria

1. Rich-text mode keeps the full grouped shadcn toolbar layout.
2. Source mode shows only the compact view-control group as a top-right overlay over the editor.
3. Diff mode shows the same compact top-right overlay over the diff editor.
4. The floating controls remain clickable and do not add a horizontal scrollbar.

## Implementation Notes

- Added mode-specific toolbar classes from MDXEditor's `viewMode$`.
- Used a floating toolbar class in source and diff modes to remove the toolbar row from normal layout.
- Kept pointer events on the floating control group while leaving the transparent overlay shell non-interactive.
