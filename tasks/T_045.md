# T_045: Add the Nexus ICO to Windows Builds

## Status

Done

## Objective

Use the project-provided Windows `.ico` file for packaged Nexus Windows artifacts.

## Scope

- Add the Windows icon path to the electron-builder Windows configuration.
- Include the `.ico` asset in packaged app file inputs.
- Document the Windows packaging icon behavior in the AI-DLC project files.

## Acceptance Criteria

- Given a Windows package is built, when electron-builder creates the executable artifacts, then it uses `nexus.ico` instead of the default Electron icon.
- Given the GitHub Actions Windows build runs, when it packages Nexus, then the Windows artifacts use the same configured icon.

## Implementation Notes

- Added `nexus.ico` to the packaged file list.
- Set `build.win.icon` to `nexus.ico`.

## Verification

- `npm run build`
- `npm run dist:win`
