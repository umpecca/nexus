# T_091: Add editor zoom controls

## Status

Done

## Summary

Add a display-only editor zoom feature with controls to zoom in, zoom out, and reset to 100%.

## Implementation Notes

- Added View menu actions for Zoom In, Zoom Out, and Reset Zoom.
- Added toolbar zoom buttons for minus, reset to 100%, and plus.
- Kept zoom as renderer-local display state so saved Markdown, profile font settings, and export typography remain unchanged.
- Applied zoom to rich-text paper sizing, margins, paragraph spacing, and source/diff font size through editor style variables.
- Added keyboard handling for `CmdOrCtrl+Plus`, `CmdOrCtrl+-`, and `CmdOrCtrl+0` in browser and Electron renderer contexts.
