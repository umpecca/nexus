# T_079: Wait for PDF preview paint before printing

## Status

Done

## Summary

Fix Print Preview PDF generation failures caused by printing the hidden export window before Chromium has completed font and paint readiness.

## Implementation Notes

- Added a shared export-document readiness wait that awaits `document.fonts.ready` and two animation frames.
- Used the readiness wait before `printToPDF` in the shared PDF renderer.
- Preserved the same PDF export and print preview rendering path.

## Verification

- Run `npm run test`.
- Run `npm run build`.
- Run `node --check electron/main.cjs`.
