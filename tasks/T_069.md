# T_069: Align export typography and PDF margin units with editor settings

**Status:** Done

**Task Description:** Make rendered HTML and PDF exports use the selected editor font family and preserve the inch-based PDF margin settings so the exported layout better matches the rich-text editor settings.

## Acceptance Criteria

1. HTML export receives the selected editor font family and base font size.
2. PDF export receives the selected editor font family, base font size, paper size, and page margins.
3. PDF export passes stored inch-based page margins to Electron `printToPDF`.
4. Invalid font family, font size, or margin values fall back to safe defaults.

## Implementation Notes

- Passed the profile-selected editor font family through the renderer export calls and preload type definitions.
- Sanitized export font families in the Electron main process before writing them into export CSS.
- Kept configured PDF margins in inches before passing them to Electron's `printToPDF` API.
