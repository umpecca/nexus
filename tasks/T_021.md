# T_021: Add a Windows-only top separator above the editor toolbar

**Status:** Done

**Task Description:** Add a subtle gray line above the editor toolbar on Windows so the toolbar visually matches its bottom separator beneath the native menu.

## Acceptance Criteria

1. Windows users see a subtle top border above the editor toolbar.
2. Non-Windows users do not receive the platform-specific top border.
3. The existing bottom toolbar separator remains unchanged.
4. Toolbar layout, wrapping, and dropdown layering remain unchanged.

## Implementation Notes

- Used the existing preload-exposed `window.nexus.platform` value to add an `app-shell-windows` class only on Windows.
- Added a scoped CSS rule that gives `.mdxeditor-toolbar` a top border only inside the Windows shell class.
- Reused the existing `--line` color so the top separator matches the bottom separator.

## Verification

- Run `npm run build`.
- Run `npm run test`.
