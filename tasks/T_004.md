# T_004-Changed Lines Review

**Task Name:** Changed Lines Review
**Task ID:** T_004
**Task Description:** Add a review panel that compares the current document to the baseline.
**Task Status:** CLOSED
**Task Status Details:** Added deterministic line comparison utilities, summary counts, and a changes tab for reviewing added, removed, and unchanged lines.

## Problem

Users need to see what changed after manual or agent-assisted edits before trusting the current document.

## Solution

1. Keep baseline Markdown alongside the current document.
2. Compare baseline and current text by line.
3. Show counts for added, removed, changed, and unchanged lines.
4. Show line-level review details in a dedicated changes mode.
5. Let users mark the current document as the new baseline.

## Verfication

Run `npm run test` after dependencies are installed. In the app, edit text after import or baseline reset and confirm the changed-lines panel updates.
