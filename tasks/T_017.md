# T_017-Remove Desktop Editor Chrome

**Task ID:** T_017

**Task Description:** Remove the outer padding and framed editor chrome so Nexus feels more like a native desktop application workspace.

**Status:** CLOSED

## Acceptance Criteria

1. The editor fills the desktop window without an outer gutter.
2. The editor surface no longer uses a framed border or rounded frame corners.
3. The focused rich-text editor no longer draws a horizontal outline that looks like the bottom of the editor.
4. The sticky toolbar and editor content remain usable after the layout change.

## Implementation Notes

- Removed app shell padding across desktop and mobile breakpoints.
- Removed the editor surface border and border radius.
- Changed the body background to match the editor paper surface.
- Removed the focused `.markdown-body` outline that created the bottom line inside the editor.

## Verification

Run `npm run build` and `npm run test`. Open the app and verify the editor is edge-to-edge and the rich-text focus outline no longer appears as a bottom boundary line.
