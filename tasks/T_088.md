# T_088: Use native spellcheck context menu

## Status

Done

## Summary

Replace the Electron desktop editor context menu path with a native Electron menu so spellcheck suggestions come directly from the same context-menu event that identifies the misspelled word.

## Implementation Notes

- Added a native Electron editable context menu with spelling suggestions, Add to dictionary, Cut, Copy, and Paste.
- Stopped using the renderer Radix context menu inside Electron windows so it cannot hide native spellcheck suggestions.
- Kept the renderer Radix context menu as a browser-only fallback for non-Electron development contexts.
- Removed the unused preload spellcheck bridge and renderer spellcheck context fetch flow.
