# T_062: Replace the Office-inspired editor toolbar with white shadcn button groups

**Task Status:** Completed

**Task Description:** Replace the gray Office-inspired MDXEditor toolbar treatment with a white shadcn-styled toolbar that organizes related commands into unlabeled button groups.

## Context

The previous toolbar used ribbon-style chrome, visible group labels, and multi-row Office-like sections. The next iteration should feel closer to the project's shadcn dialogs and controls while preserving the broad Markdown command set.

## Acceptance Criteria

1. The editor toolbar background is white.
2. Toolbar commands are visually organized into button groups.
3. Button groups do not show visible text labels.
4. View mode and paper view controls remain accessible from the toolbar.
5. Existing MDXEditor formatting, insert, block, and view commands remain available.

## Implementation Notes

- Added a local shadcn-style `ButtonGroup` primitive.
- Replaced the ribbon wrapper and labeled sections with unlabeled command groups.
- Converted view and paper controls to icon-only shadcn buttons with tooltips and ARIA labels.
- Updated toolbar styling from gray ribbon chrome to a white command bar.

## Verification

Run `npm run build`.
