# T_023: Add a shadcn-styled editor context menu

**Status:** Done

**Task Description:** Add a right-click context menu to the editor surface using shadcn-style context menu components, with Cut, Copy, and Paste commands.

## Acceptance Criteria

1. Right-clicking inside the editor opens a project-styled context menu.
2. The context menu includes Cut, Copy, and Paste.
3. Cut, Copy, and Paste operate on the current editor selection or caret.
4. The menu uses local shadcn-style components backed by Radix primitives.
5. The implementation keeps direct clipboard access out of renderer code.

## Implementation Notes

- Added `@radix-ui/react-context-menu`.
- Added `src/components/ui/context-menu.tsx` as the local shadcn-style context menu primitive.
- Added `src/components/editor/EditorContextMenu.tsx` around the MDXEditor surface.
- Added an Electron `edit:command` IPC handler for standard Cut, Copy, and Paste actions against the active web contents.
- Restored the saved editor selection before dispatching context-menu edit commands.

- Run `npm run build`.
- Run `npm run test`.
