# T_048: Exit Lists From Empty List Items

## Goal

Make list editing behave like a regular desktop editor: after creating a blank list item, pressing Enter again should exit the list and create a normal paragraph.

## Scope

- Preserve normal Enter behavior for non-empty list items.
- Preserve Shift+Enter and modifier-key shortcuts.
- Support bullet, numbered, and checklist items through the existing MDXEditor list plugin.
- Keep the fix local to rich text mode.

## Implementation Notes

- Added a small MDXEditor realm plugin that registers a high-priority root Lexical Enter command.
- The plugin detects collapsed selections inside empty list items, including empty child paragraph representations.
- When matched, it clears the empty list item, selects it, and delegates to Lexical's list paragraph insertion helper.

## Acceptance Criteria

- Typing a bullet item and pressing Enter creates the next list item.
- Pressing Enter again on the blank list item exits the list.
- Pressing Enter in a non-empty list item continues the list.
