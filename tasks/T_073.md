# T_073: Add a packaged macOS app bundle icon

## Status

Done

## Summary

Add a generated macOS `.icns` asset and configure Electron Builder to use it for packaged Nexus macOS app bundles so Finder and Dock show the Nexus icon instead of Electron's default icon.

## Implementation Notes

- Added a cross-platform `generate:mac-icon` package script.
- Added `scripts/generate-mac-icon.mjs`, which uses `sharp` to resize the root `nexus.png` asset and write a valid `nexus.icns` container.
- Added `nexus.icns` to packaged files and set `build.mac.icon` to `nexus.icns`.
- Kept the existing runtime `app.dock.setIcon(nexus.png)` fallback for development launches.

## Verification

- Run `npm run generate:mac-icon` and confirm `nexus.icns` is created.
- Run `npm run test`.
- Run `npm run build`.
- Run `git diff --check`.
- On macOS, run `npm run dist:mac` and launch the packaged app to confirm the Dock icon.
