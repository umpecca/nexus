# T_083: Harden PDF preview generation against hidden-window print failures

## Status

Done

## Summary

Fix Print Preview and PDF export failures when Electron's hidden Chromium print renderer rejects every configured print attempt.

## Implementation Notes

- Keep Print Preview and Export as PDF on the shared `renderMarkdownPdf` path.
- Use Electron PDF margin options in inches for the fallback `printToPDF` call.
- Make hidden export windows more deterministic for paint and print readiness.
- Add a final off-screen inactive window print attempt after hidden-window retries fail.
- Return a concise final error message when all PDF generation attempts fail.

## Verification

- Run `node --check electron/main.cjs`.
- Run `npm run test`.
- Run `npm run build`.
- Manually verify File/Print Preview, Refresh, Save PDF, and File/Export as PDF in Electron.
