# T_049: Add HTML and PDF Export

## Goal

Add File menu actions that export the current Markdown buffer as rendered HTML or PDF.

## Scope

- Add `File > Export as HTML`.
- Add `File > Export as PDF`.
- Use native save dialogs for export destinations.
- Render exports from the current in-memory editor buffer.
- Resolve local relative Markdown image paths relative to the current Markdown file path when available.
- Do not change the current file path, saved baseline, or dirty state.

## Implementation Notes

- Added focused-window menu actions for `exportHtml` and `exportPdf`.
- Added preload IPC for HTML and PDF export.
- Added main-process export handlers that render Markdown with Marked.
- Reused the local image preview resolver to turn relative local Markdown image sources into file URLs for exported documents.
- PDF export loads the generated HTML into a hidden Electron window and writes `printToPDF` output.

## Acceptance Criteria

- Selecting File/Export as HTML prompts for an `.html` destination and writes a rendered HTML document.
- Selecting File/Export as PDF prompts for a `.pdf` destination and writes a PDF.
- Exporting a dirty buffer includes unsaved in-memory edits without marking the document saved.
- Relative local images in Markdown export correctly when the document has a current file path.
