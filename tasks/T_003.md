# T_003-File Import And Export

**Task Name:** File Import And Export
**Task ID:** T_003
**Task Description:** Add local Markdown import and export behavior.
**Task Status:** CLOSED
**Task Status Details:** Added file input import, Markdown export as a downloaded file, and baseline updates on import and export.

## Problem

Nexus must work with existing Markdown files and let users save their edited Markdown outside the app.

## Solution

1. Add a hidden file input that accepts Markdown files.
2. Read selected files as text.
3. Store imported content as the current document and baseline.
4. Export the current document as a `.md` file.
5. Update the baseline after export.

## Verfication

Run `npm run dev`, import a `.md` file, edit it, export it, and confirm the downloaded Markdown contains the edits.
