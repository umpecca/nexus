# T_089: Add in-editor text find

## Status

Done

## Summary

Add a desktop-style Find command that lets users locate text in the current editor content without leaving the document surface.

## Implementation Notes

- Added an Edit/Find menu item with the standard `CmdOrCtrl+F` accelerator.
- Added a renderer find panel backed by MDXEditor's search plugin.
- Escaped user-entered find text before passing it to MDXEditor's regex-based search query so the panel behaves like literal text search.
- Added previous and next match navigation, match counts, and visible search/focused-match highlights.
