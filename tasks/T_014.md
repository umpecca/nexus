# T_014-Electron Edit Menu Actions

**Task ID:** T_014

**Task Description:** Add Undo, Redo, Copy, and Paste support through a native Electron Edit menu.

**Status:** CLOSED

## Acceptance Criteria

1. The Electron app menu includes a top-level Edit menu.
2. The Edit menu includes Undo, Redo, Copy, and Paste items.
3. The menu items use native Electron edit behavior for the currently focused editor control.

## Implementation Notes

- Added a top-level Edit menu in the Electron main-process menu template.
- Used Electron built-in menu roles for `undo`, `redo`, `copy`, and `paste`.
- Kept clipboard and editing behavior out of preload IPC because Electron roles already dispatch these commands safely.

## Verification

Run `npm run build` and `npm run test`. Launch the Electron app and manually verify the Edit menu items while focus is inside the MDXEditor rich text and source editors.
