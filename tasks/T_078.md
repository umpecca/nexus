# T_078: Add PDF print preview from the File menu

## Status

Done

## Summary

Add a File menu print preview workflow that renders the current Markdown document through the same PDF export path and displays it before saving.

## Implementation Notes

- Added File/Print Preview as a focused renderer menu action.
- Refactored PDF export rendering into a shared Electron helper that returns PDF bytes.
- Added preload IPC for generating preview bytes and saving an already generated preview PDF.
- Added a shadcn-styled print preview dialog with loading, error, refresh, save, and close states.

## Verification

- Run `npm run test`.
- Run `npm run build`.
- Verify File/Print Preview opens a PDF snapshot, Refresh regenerates it, and Save PDF writes the previewed PDF.
