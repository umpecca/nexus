# T_006-Use MDXEditor Source Toggle

**Task Name:** Use MDXEditor Source Toggle
**Task ID:** T_006
**Task Description:** Remove the custom Visual/Source toggle and use MDXEditor's built-in source mode toolbar control.
**Task Status:** CLOSED
**Task Status Details:** Added MDXEditor's diff/source plugin and toolbar wrapper with rich text and source options only. Removed the custom mode state, tab bar, plain textarea source editor, and related styles.

## Problem

The app had its own Visual/Source toggle and source textarea even though MDXEditor already provides source editing through its toolbar. This duplicated behavior and made the editor surface feel split between app-level and editor-level controls.

## Solution

1. Add `diffSourcePlugin` to the MDXEditor plugin list.
2. Wrap toolbar controls in `DiffSourceToggleWrapper`.
3. Limit the MDXEditor toggle options to rich text and source mode.
4. Remove custom editor mode state, mode tabs, and textarea source editor.
5. Update product and architecture docs to describe source editing as an MDXEditor toolbar capability.

## Verfication

Run `npm run build` and `npm run test`. Open the app and confirm there is no separate Visual/Source tab bar and that source mode is available from the MDXEditor toolbar.
