# T_071: Add bundled web font choices

**Status:** Done

**Task Description:** Add a curated set of locally bundled web fonts to the editor font preferences and make HTML/PDF exports use the same selected font.

## Acceptance Criteria

1. Preferences lists Roboto, Open Sans, Lato, Source Sans 3, Merriweather, Source Serif 4, JetBrains Mono, and Roboto Mono.
2. The renderer loads regular, italic, and bold local font files for each bundled font.
3. HTML and PDF exports allow the new font-family values and inject local Fontsource CSS for bundled web fonts.
4. Unknown stored font values still fall back to the default editor font.
5. Tests and production build pass.

## Implementation Notes

- Installed Fontsource packages for the curated font set plus Inter for the existing default stack.
- Imported Latin Fontsource CSS in the renderer entrypoint for editor and Preferences rendering.
- Mirrored the selectable font allowlist in Electron main and added export CSS import generation through `require.resolve`.
