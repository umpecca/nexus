# T_085: Remove PDF print preview

## Status

Done

## Summary

Remove the File/Print Preview workflow because Electron PDF preview generation remains unreliable on the target desktop runtime. Keep File/Export as PDF as the supported PDF output path.

## Implementation Notes

- Removed the File/Print Preview native menu item and renderer menu action handling.
- Removed the print preview dialog component, state, refresh handling, and save-preview handling.
- Removed preview-only preload methods, global TypeScript types, and Electron IPC handlers.
- Kept File/Export as PDF and its existing PDF generation fallbacks intact.
- Updated product, architecture, and changelog documentation to describe PDF export without print preview.
