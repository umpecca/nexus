# T_087: Fix spellcheck context menu timing

## Status

Superseded by T_088

## Summary

Fix the editor right-click menu so spelling suggestions appear for misspelled words that Electron already underlines.

This timing-based renderer bridge did not resolve the desktop behavior consistently. T_088 replaces the Electron desktop path with a native Electron context menu.

## Implementation Notes

- Cached the latest Electron spellcheck context per renderer window in the main process.
- Added a preload getter so the renderer can request the latest spellcheck context for the current right-click.
- Updated the editor context menu to fetch spellcheck context shortly after right-click, retry briefly for Electron event ordering, and ignore stale contexts.
- Cleared spellcheck suggestions when the context menu closes so suggestions do not appear on later non-misspelled words.
