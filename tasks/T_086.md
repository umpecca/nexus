# T_086: Add editor spell checking

## Status

Done

## Summary

Add native Electron spell checking to the editor so misspelled words show inline browser underlines and the existing right-click menu offers corrections.

## Implementation Notes

- Enabled Electron spellcheck for editor windows.
- Routed Electron spellcheck context data through the preload bridge.
- Added spelling suggestions and Add to dictionary to the editor context menu while keeping Cut, Copy, and Paste.
- Used Electron's built-in misspelling replacement and user dictionary APIs instead of exposing clipboard or document text to the renderer.

- Run `npm run test`.
- Run `npm run build`.
