# T_013-Sticky MDXEditor Toolbar

**Task ID:** T_013

**Task Description:** Keep the MDXEditor toolbar sticky at the top of the editor while reserving the remaining editor frame for the editable document area.

**Status:** CLOSED

## Acceptance Criteria

1. The MDXEditor toolbar stays visible at the top of the editor frame.
2. The document editing area receives the remaining vertical space and scrolls independently.
3. The toolbar does not cover or overlap editable document content.

## Implementation Notes

- Converted the editor surface into a full-height flex frame.
- Made the MDXEditor root a column layout with the toolbar above the scrollable editor content.
- Kept the toolbar sticky with a solid background and border so it remains visually distinct while content scrolls below it.

## Verification

Run `npm run build` and inspect the app with a long document.
