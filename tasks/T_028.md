# T_028: Add a Help/About dialog

**Task Description:** Add a native Help menu with an About item that opens a shadcn-styled modal dialog.

**Task Status:** Completed

**Task Status Details:** Added Help/About to the Electron menu, forwarded the action to the renderer, and added a reusable About dialog that displays the requested title and copyright text.

## Product Context

Nexus is growing into a traditional desktop application. A Help/About entry gives the app a standard place for application ownership and copyright information without adding controls to the editor workspace.

## Acceptance Criteria

1. The native menu includes Help.
2. The Help menu includes About.
3. Clicking About opens a shadcn-styled modal.
4. The modal title is `About`.
5. The modal body shows `Copyright 2026 Vince`.
6. The app build and test suite continue to pass.

## Implementation Notes

- Added the Help/About menu item in `electron/main.cjs`.
- Extended renderer menu action typing with `about`.
- Added `src/components/about/AboutDialog.tsx` using the existing local dialog primitive.
- Updated `src/App.tsx` to open and close the About dialog.

## Verification

Run `npm run build` and `npm run test`. Launch the app and confirm Help/About opens the modal with the requested title and body text.
