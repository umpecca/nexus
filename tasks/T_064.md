# T_064: Match the editable page background to the toolbar background

**Task Status:** Completed

**Task Description:** Make the rich-text editable page background use the same color token as the toolbar so the editing surface and toolbar stay visually consistent.

## Context

The toolbar now uses a white shadcn-styled background. The editable page and plain-mode editor surface should reference the same toolbar background token instead of separate hard-coded or off-white colors.

## Acceptance Criteria

1. The paper-view editable page background matches the toolbar background.
2. The plain-view editor background matches the toolbar background.
3. The implementation uses the shared toolbar color token so future toolbar color changes stay synchronized.

## Implementation Notes

- Changed the plain-view rich-text editor background to `var(--toolbar)`.
